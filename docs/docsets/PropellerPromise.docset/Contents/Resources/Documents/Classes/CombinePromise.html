CombinePromise

public class CombinePromise: Promise<[Any]>

Promise subclass that is initialized with other promises to complete. CombinePromise fires failure then or complete after all of these promises are fullfilled/rejected. If one or more of the waiting promises are rejected, this CombinedPromnise is rejected. Promises can only be used with one CombinedPromise.

  • init(promises:)

    Initializer for CombinedPromise requiring list of other Promisables to wait on before being fired. If one or more of these Promisable values are rejected this CombinedPromise is also rejected. The protocol Promisable instead of the Promise class is used here to allow Promise objects with mixed associated values to be specified.

    • Parameters:
      • promises: Array of Promisable values (usually Promise objects) that must be fullfilled/rejected before this CombinedPromise will become fullfilled/rejected.
